# 随笔

这里放一些不需要写成完整教程或正式文章的内容：突然想到的问题、阶段性的理解、生活观察，以及值得记住的小事。

## 会写什么

- 某个阶段结束后的复盘。
- 学习和生活习惯的变化。
- 对一件小事的观察。
- 当时还没有答案的问题。
- 想留给未来自己的几句话。

## 写作原则

随笔可以短，也允许保留不确定性。重要的是准确记录当时真实的想法，而不是强行得出一个完整结论。

    新随笔会按时间添加，内容增多后再按年份整理。
