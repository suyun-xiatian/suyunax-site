# 足迹

这里记录我去过的地方。它不只是一份地点清单，也会保存路线、照片、当时的感受，以及之后回想起来仍然清晰的片段。

## 记录方式

每段足迹可以包含：

- 出发时间和目的地。
- 实际走过的路线。
- 值得停留的地点和体验。
- 交通、住宿与费用参考。
- 照片和一句话回忆。
