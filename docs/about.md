# 关于本站和我

这里是我的个人网站。它会慢慢收集我在学习、写代码、看球、旅行和日常生活中的记录：有些内容会像笔记一样严肃，有些会像日记一样松弛。总之，这是一个把零散想法安放下来的地方。

## 我是谁

我是夏天，目前主要关注统计、数据分析和编程，也喜欢足球、旅行和记录生活。这个网站不是一份正式简历，更像一张持续更新的个人地图：我学过什么、正在理解什么、哪些事情让我觉得值得写下来。

## 为什么做这个网站

我希望它承担三个作用：

- **整理知识**：把统计、数学、编程中的概念写成自己能复述的版本。
- **记录过程**：保存项目、课程、旅行和生活里的阶段性材料。
- **公开输出**：让以后回头看时，能看到自己是怎样一点点变清楚的。

参考站点给我的启发是：个人网站不必一开始就很庞大，但最好有稳定的栏目、清楚的入口，以及一种愿意长期维护的秩序感。

## 这个网站会写什么

目前先分成几块：

| 栏目 | 内容 |
| --- | --- |
| 学习笔记 | 统计、概率、数学方法、课程复盘 |
| 编程 | Python、数据分析、网页搭建、工具使用 |
| 生活 | 足球、旅行足迹、随笔、读书观影和日常观察 |
| 更新日志 | 记录网站结构和内容的变化 |

## 写作原则

- 能讲清楚的地方尽量讲清楚，不为了显得复杂而复杂。
- 重要结论尽量配例子、代码或推导。
- 允许保留阶段性理解，但会在之后修正。
- 文章可以短，但最好有一个明确的问题意识。
